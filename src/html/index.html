Инструкция по установке и использованию Outline VPN

Выберите вашу платформу, установите клиент и вставьте ключ доступа.

Windows Linux Android iOS macOS

Шаги установки

  1. Скачайте Outline Client для вашей платформы (ссылки выше).
  2. Откройте приложение и вставьте ключ доступа, полученный в Telegram-боте.
  3. Нажмите «Подключиться» — соединение установится автоматически.
Подсказка: если ключ вставился не полностью, попробуйте ещё раз — клиент принимает ключ целиком, включая префикс ss:// или outline://.

Вопросы по установке или доступу — пишите в поддержку.
